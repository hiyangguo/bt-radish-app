# BT 萝卜
BT 萝卜是一款使用 Electron 构建的桌面磁力搜索软件。

## 搜索源
- [x] [BT Book][btbook-home]
- [x] [屌丝搜][diaosiso-home]

## 文档
[API 设计文档](./docs/API.md)

## 开发运行
1. 安装
```bash
npm install
```
2. 其他命令
```bash
## 启动 electron
npm run start
## 启动 web 开发服务
npm run start-dev
## 打包
npm run packager
```
